* penjelasan : Kode program HTML ini digunakan untuk membuat tampilan website sederhana berupa Sistem Manajemen Sepatu yang berfungsi sebagai dashboard untuk mengelola data produk sepatu. Website ini menampilkan beberapa bagian utama yaitu navbar sebagai header navigasi, hero section sebagai banner utama yang berisi judul sistem, dashboard statistik untuk menampilkan jumlah produk, stok tersedia, dan kategori, daftar sepatu yang ditampilkan menggunakan card Bootstrap, serta form input untuk menambahkan data sepatu baru. Program ini memanfaatkan framework Bootstrap melalui CDN untuk mempermudah pembuatan tampilan yang responsif dan rapi dengan menggunakan komponen seperti grid system (container, row, col), card, button, dan form. Selain itu, terdapat CSS custom yang digunakan untuk mengatur tampilan tambahan seperti background hero, font, warna halaman, serta efek hover pada dashboard card. Secara keseluruhan, kode ini berfungsi sebagai antarmuka (frontend) sistem manajemen sepatu yang memudahkan pengguna dalam melihat informasi produk dan melakukan input data secara sederhana.
